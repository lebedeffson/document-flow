# Legacy Materials

Сюда вынесены старые архивы, PDF-мануалы, дистрибутивы и служебные отчеты,
которые полезно хранить рядом с проектом, но не нужно держать в корне репозитория.

Структура:

- `vendor/` — старые дистрибутивы, архивы, PDF и ebuild-файлы
- `reports/` — старые отчетные материалы по `NauDoc`
